To use the typelibrary and run the demo,

1) Copy WinApiForVb.tlb located in WinApiForVb_Compiled folder to your system directory.

2) Open up the AllApiDemo project and goto references and use the Browse button to manually locate and select the WinApi For VB Typelibrary. Click ok and goto object browser, you should be able to see the contents of the library.


=========================================================
The AllApiDemo demonstrates how to create and use (using only API); message loops with idle handling, forms (Model/Modeless,various styles,own msg loop), Animation, Button (various styles, ComboBoxEx (image), ComboBox (Drive/Ownerdrawn+images)  Date and Time Picker (getting date+limiting range), Edit (various styles-num/text/LCase/UCase), Hot Key, Image List (Loading icons+bitmaps+imagestrips+drawing), Ip Address (limiting range), ListBox (Standard+Ownerdraw+Chkimages) , ListView (images+owenerdrawn+Edit control+Tooltips+Contextmenu), Month Calendar, Pager, Progress Bar (Various styles), Rebar (toolbar+inplace comboBox), RichEdit (v2.0+Contextmenu), Static (Label/image/frame), Status (various style panes+image+embedded progress bar, Tab (image+tooltips), Toolbar (sysimages+various btn styles+tooltips), Tooltip (normal/baloon), Trackbar, Treeview (image+stateimage+Ownerdrawn+Fonts+Contextmenu+tooltips+dragdrop+Save+Load), UpDown (buddy+limiting range), Timer (using the same wndproc as the forms, no extra moduel for timer. If you ever subclass a form and at the same time used an api timer, you will see here that you can use the same wndproc to process the timer msgs without using an extra module or routine).
